Who Am I?

Problem Solving with Programs

I didn't know I wanted to be a software developer...

But I did want to be an engineer... When I started my collegiate career I wanted to be a mechanical engineer. Unfortunately that did not work out. I was so discouraged because I was sure that Mehcanical Engineering was for me. I ended up taking some computer science classes just to see if CS would be a possible alternative. With no prior coding experience, I found the classes quite difficult. But, I learned about the plethora of things that could be built with software and I was hooked! Dispite the fact I later had to switch out of the CS program due to health reasons, I retained my passion for software devlopment and I am always looking at ways to improve my technical skills and to apply them to problems in my everyday life.
