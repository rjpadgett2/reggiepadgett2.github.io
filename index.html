iAdvisor

The Online Advising System

General Description

iAdvisor is an online 4 year plan building system that uses the students past class history to build a 4 year plan. The program will automatically check for cases like too many credits, too little credits, ensure the correct class order, and keep track of classes needed to complete thier degree. The students will be able to send thier agenda in a CSV format to advisors in the iSchool at the University of Maryland College Park so it can be inserted into thier database. The goal for this project is to make managing multiple student 4 year plans easier for advisors and make creating 4 year plans less complicated for the students.

Technology Used

The program uses JavaScript, JQuery, Bootstrap, HTML, and CSS for the front end interface as well as form and 4 year plan builder. PHP is used on the back-end to process request and MySQL is used as the database. We are using the Google Sign-In API to keep track of users and their state. We are using the Google Clooud platform to host the website.
